# auction_house
A simpe auction house (AngularJS, JavaEE)
A simple auction house, following ebay's profile. The site had three user types. Guest, who was only able to view the auctions available. Registered user, who could buy, post and bid on an auction. Administrator who apart from the above-mentioned, could also approve user's request for registration, download and view an auction in an XML file, and upload one or more auctions from an XML file. Technologies used: AngularJS, Java, HTML, CSS, MySQL, XML
