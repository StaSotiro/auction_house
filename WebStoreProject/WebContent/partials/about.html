Greek Bay

To Greek Bay είναι το μεγαλύτερο auction site στην Ελλάδα.

Δημιουργήθηκε από τους Λεωτσαράκο Δημήτριο και Σταύρο Σωτηρόπουλο, φοιτητές στο τμήμα Πληροφορικής και Τηλεπικοινωνιών του Εθνικού Καποδιστριακού Πανεπιστημίου.

Η ομάδα μας

Λεωτσαράκος Δημήτριος‚ Web Developer

Ο Δημήτριος Λεωτσαράκος είναι ο Frontend Developer. Ασχολήθηκε κυρίως με την AngularJS και την δημιουργία των σελίδων με HTML και το Bootstrap Framework.

Σταύρος Σωτηρόπουλος‚ Web Developer

Ο Σταύρος Σωτηρόπουλος είναι ο Backend Developer. Ασχολήθηκε κυρίως με τη λειτουργία του server σε Java και τη βάση δεδομένων(JPA, Restful Web Services)
